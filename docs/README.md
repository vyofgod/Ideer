# Documentation

This directory contains project documentation and product planning notes.

## Structure

- `src/`: user-facing mdBook documentation inherited from upstream Zed.
- `product/`: Ideer-specific product, release, and publishing planning.
- `theme/`: mdBook theme assets.

Before publishing Ideer publicly, review the upstream Zed-specific pages under `src/` and either rewrite, remove, or clearly mark them as inherited upstream documentation.
